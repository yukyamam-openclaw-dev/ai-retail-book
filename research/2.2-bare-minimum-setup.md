# Research: 2.2 Bare Minimum Setup

## Goal
Define the absolute minimum requirements for a non-technical retail owner to start their 'AI Store Manager'.

## Key Components & Requirements
1. **Hardware/Hosting Options**
   - **Mac (Local)**: Apple Silicon (M1/M2/M3) is ideal due to Unified Memory. Minimum 16GB RAM recommended.
   - **VPS (Cloud)**: Ubuntu 22.04 LTS. Minimum 2 vCPUs and 8GB RAM (though 16GB+ is safer for LLMs). GPU is optional if using small models or API bridging.

2. **Software Stack**
   - **Ollama**: The engine for running LLMs locally. simple install, easy model management.
   - **Hermes Agent**: The orchestration layer. Needs Python environment.
   - **Docker (Optional but Recommended)**: To keep the environment isolated and easy to migrate.

3. **Baseline Model Selection**
   - **Llama 3 / Mistral / Gemma**: Small but capable models that can run within 8-16GB RAM.
   - **Fine-tuning vs. RAG**: Focus on RAG (Retrieval Augmented Generation) using a simple knowledge base (Markdown files/Notion) instead of complex fine-tuning.

4. **Connectivity**
   - Static IP or Dynamic DNS for the VPS to ensure the agent can be reached by external webhooks (e.g., SNS, Shopify).

## Implementation Plan for the Store Owner
- Step 1: Install Ollama.
- Step 2: Download a recommended 'base' model.
- Step 3: Install Hermes Agent.
- Step 4: Configure basic memory (Store rules).
