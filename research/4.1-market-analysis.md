# Research: 4.1 【市場分析】競合店とトレンドの自動監視

## 目的
小売店主が競合店の動向や市場トレンドを「自律型AIエージェント」を用いて自動的に収集・分析し、意思決定に活かす手法を定義する。

## 調査事項
1. **監視対象の特定**: 
    - 競合店のWebサイト、SNS（X, Instagram）、Googleマップのレビュー、業界ニュース。
    - Amazonや楽天などのECプラットフォームにおける類似商品の価格変動。
2. **自律エージェントによる自動化フロー**:
    - `web_search` または `browser` ツールを用いて定期的に特定URLを巡回。
    - 変更点（新商品、価格改定、キャンペーン開始）を抽出。
    - まとめをNotionやSlackに通知。
3. **分析の視点**:
    - 「競合が何を始めたか」だけでなく、「なぜ今それを始めたか」をAIに考察させる（市場トレンドとの照合）。
4. **World Wide Insight (English Sources)**:
    - 海外のAI Agent Frameworks (AutoGPT, CrewAI) を使ったマーケットインテリジェンスの事例。
    - Perplexity API を用いた競合分析の自動化ワークフロー。
